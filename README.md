# EDAM_ground_truth

Ground truth dataset of life sciences software, including 45 analyzed tools.
The annotations are compared with those in [bio.tools registry](https://bio.tools/) using the [**EDAM ontology**](https://edamontology.org/page).
This ground truth are supported by the [ShareFAIR](https://projet.liris.cnrs.fr/sharefair/).

In this repo, we provide a list of bioinformatics tool annotations generated by a community of life science experts, covering the field of 
- Genetic_variant
- Microscopy
- Neuroimaging
- Phylogeny
- Single_Cell
- Systems_biology

Annotations generated by an LLM (DeepSeek V3.1) were also provided and either validated or rejected by the experts. 
The consensus on these annotations between the experts and the LLM is also made available.

## Content 

This repository contains resources related to the EDAM ground truth dataset of life sciences software. 
Specifically, it includes:

- [`Prompt_examples_two_annotated_tools.txt`](Prompt_examples_two_annotated_tools.txt) – Examples of prompts provided to DeepSeek V3.1 to generate annotations following the same procedure as domain experts.
- [`Confusion_matrix_free_text_annotation/`](Confusion_matrix_free_text_annotation/) – Confusion matrices comparing annotations produced by the LLM and experts for each scientific domain. 
  - These matrices were computed using the notebook [`Confusion_matrix_json.ipynb`](Confusion_matrix_json.ipynb).
- [`Contingency_table/`](Contingency_table/) – Contingency tables of free-text annotations validated by experts, showing results compared to existing entries in the bio.tools database.
- [`Distribution_free_text_annotation/`](Distribution_free_text_annotation/) – Free-text annotations organized by type, including topic, operation, input/output data type, and input/output format. 
  - These distributions were calculated and visualized using [`Distribution_free_text_annotation.ipynb`](Distribution_free_text_annotation.ipynb).
- [`Raw_free_text_annotation/`](Raw_free_text_annotation/) – Raw annotations from both experts and the LLM, along with their conversion into JSON format.
- [`plots/`](plots/) – Figures used in the associated publication.

## Usage

- **Explore annotations:** Examine the JSON and free-text annotation files in [`Raw_free_text_annotation/`](Raw_free_text_annotation/).
- **Analyze distributions:** Run [`Distribution_free_text_annotation.ipynb`](Distribution_free_text_annotation.ipynb) to reproduce annotation distributions and visualizations.
- **Evaluate model performance:** Use [`Confusion_matrix_json.ipynb`](Confusion_matrix_json.ipynb) to compute confusion matrices comparing LLM and expert annotations.
- **Compare with bio.tools:** Refer to the [`Contingency_table/`](Contingency_table/) for expert validation against existing database entries.
